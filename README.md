# HuntFlow Map Editor

A simple map editor for [HuntFlow](https://github.com/GodXero33/HuntFlow).  
Used to design and test custom maps for the game.

## Features

- Draw and adjust boundaries
- Add map objects
- Export map data for HuntFlow
